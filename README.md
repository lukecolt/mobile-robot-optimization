## Running application
- Clone repo
- Open cmd prompt in working directory
- Run command:
  ```
  pip install -r requirements.txt
  ```
- To run app, write following command in cmd prompt
  ```
  python main.py
  ```
- Then wait a moment, after few seconds you should see app
